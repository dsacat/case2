# Командный кейс № 2 «Управление столовой»

Веб-приложение «Умная столовая» для автоматизации работы школьной столовой.

## Возможности
- Каталог блюд с группами, фото, составом и ценой
- Оформление и выдача заказов
- Родительский контроль: привязка ребёнка, лимиты трат, запрещённые продукты
- Платежи и баланс
- Интерфейс кухни: выдача, склад, заявки на закупку, инциденты
- Обратная связь (тикеты)
- Уведомления
- Отчёты с графиками и экспортом в CSV
- Тёмная и светлая тема
- Административная консоль (CLI + веб)

## Стек
- Python 3.10+, Flask, SQLAlchemy, SQLite
- Jinja2, CSS3, Vanilla JS
- Pillow (обработка изображений, AVIF)

## Быстрый старт

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python .\site\main.py
```

Открыть `http://127.0.0.1:5000`. При первом запуске — мастер настройки на `/setup/`.

## Роли
| Роль | Доступ |
|------|--------|
| student | Меню, заказы, профиль, баланс |
| parent | + управление ребёнком, ограничения |
| moder | + модерация обращений |
| chef | + кухня, склад, выдача |
| admin | + управление пользователями, настройки |
| super_admin | Полный доступ, консоль |
