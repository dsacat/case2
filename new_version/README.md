# Командный кейс №2: «Управление столовой»

Веб-приложение для автоматизации школьной столовой: меню, заказы, баланс, выдача, обратная связь и отчетность.

## Основные возможности

- Каталог блюд с фото, составом, КБЖУ и ценой
- Обычные заказы и предзаказы
- Роли пользователей: школьник, родитель, модератор, повар, администратор
- Родительский контроль: привязка ребенка, лимиты, блокировки блюд и аллергенов
- Баланс, операции и история пополнений
- Модуль кухни: выдача, склад, закупки, инциденты
- Обратная связь и уведомления
- Отчеты и экспорт CSV

## Технологии

- Python 3.10+
- Flask
- SQLAlchemy + SQLite
- Jinja2, CSS, JavaScript
- Pillow + pillow-avif-plugin

## Быстрый запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python .\site\main.py
```

После запуска откройте:

- `http://127.0.0.1:5000`

При первом старте используйте мастер начальной настройки:

- `http://127.0.0.1:5000/setup/`

## Структура проекта

```text
site/
  main.py                 # Точка входа
  routes/                 # Маршруты приложения
  templates/              # HTML-шаблоны
  static/                 # CSS, JS, изображения

```
