# Инструкция по запуску

## 1. Требования

- Windows 10/11
- Python 3.10 или выше
- Доступ к интернету для установки зависимостей

## 2. Установка зависимостей

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## 3. Запуск приложения

```powershell
python .\site\main.py
```

После запуска открыть:

- `http://127.0.0.1:5000`

## 4. Первый запуск

Если база пустая, откроется мастер начальной настройки:

- `http://127.0.0.1:5000/setup/`

В мастере задаются параметры проекта и создается учетная запись администратора.

## 5. Основные роли

- `student` — заказы, профиль, баланс
- `parent` — управление ребенком и лимитами
- `moder` — работа с обращениями
- `chef` — модуль кухни и выдача
- `admin` / `super_admin` — администрирование
